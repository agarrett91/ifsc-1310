    To properly elaborate thoughts on the topic, I first have to ask "What is progressive enchancement?" As Mozilla describes it, "Progressive enhancement is a design philosophy that provides a baseline of essential content and functionality to as many users as possible, while delivering the best possible experience only to users of the most modern browsers that can run all the required code."
